# Amazon Worker Reviews – Exploratory Data Analysis (EDA)

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on Amazon worker reviews to understand employee sentiment, workplace satisfaction, and key factors influencing reviews across **time, roles, and countries**.

The analysis focuses on identifying trends in ratings, pay, work-life balance, job security, and overall employee experience using structured data and visual analytics.

---

## Key Objectives
- Analyze **year-wise trends** in employee ratings and metrics  
- Compare **country-wise differences** in worker satisfaction  
- Explore relationships between **pay, benefits, work environment, and job security**  
- Extract insights from **textual reviews** using word clouds  
- Summarize patterns that reflect Amazon’s internal work culture  

---

## Tools & Technologies
- **Python**
- **Pandas, NumPy** – data manipulation  
- **Matplotlib, Plotly** – visualization  
- **Streamlit** – interactive dashboard  
- **WordCloud** – text analysis
